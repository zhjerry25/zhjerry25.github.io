## 一、标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

> 注：`#` 与标题文字之间建议保留空格

---

## 二、段落与换行

- 段落：直接连续文字，空行分隔不同段落
- 换行：行末加 **两个空格** 后回车，或使用 `<br>`

---

## 三、强调

| 效果         | 语法                  |
| ---------- | ------------------- |
| **粗体**     | `**粗体**` 或 `__粗体__` |
| *斜体*       | `*斜体*` 或 `_斜体_`     |
| ***粗斜体***  | `***粗斜体***`         |
| ~~删除线~~    | `~~删除线~~`           |
| <u>下划线</u> | `<u>下划线</u>`        |

---

## 四、列表

### 无序列表

```markdown
- 项目 1
- 项目 2
  - 子项目（缩进 2 或 4 空格）
* 星号也可
+ 加号也可
```

### 有序列表

```markdown
1. 第一项
2. 第二项
   3. 子项（缩进）
4. 第三项
```

### 任务清单

```markdown
- [x] 已完成
- [ ] 未完成
- [ ] 待办
```

---

## 五、链接与图片

### 链接

```markdown
[显示文字](https://example.com)
[带标题](https://example.com "鼠标悬停显示")
```

### 图片

```markdown
![替代文字](https://example.com/image.jpg)
![图片](img.png "可选标题")
```

### 引用式链接（复用）

```markdown
[文字][id]
[id]: https://example.com
```

## 六、引用

```markdown
> 这是引用
> 多行引用
>> 嵌套引用
```

---

## 七、代码

### 行内代码

```markdown
`console.log('hello')`
```

### 代码块（围栏式）

````markdown
```python
print("Hello")
```
````

> 支持指定语言（如 `python`、`javascript`、`bash` 等）实现语法高亮

### 缩进式代码块（每行前 4 空格或 1 Tab）

```markdown
    def hello():
        print("Hi")
```

---

## 八、表格

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | ------: |
| 内容   |  内容    |   内容  |
| 行2    |  行2     |  行2    |
```

> `---` 表示表头分隔，`:` 控制对齐方向

---

## 九、分隔线

```markdown
---
***
___
```

> 单独一行，至少 3 个符号

---

## 十、转义字符

使用反斜杠 `\` 转义 Markdown 特殊字符：

```markdown
\* 不是斜体
\# 不是标题
\[ 不是链接
```

---

## 十一、HTML 标签（扩展）

多数 Markdown 渲染器支持内联 HTML，可用于复杂排版：

```html
<span style="color:red">红色文字</span>
<br> 换行
<kbd>Ctrl</kbd> + <kbd>C</kbd>
```

---

## 十二、其他常用扩展（部分平台）

### 1. 脚注

```markdown
这是带脚注的文字[^1]
[^1]: 脚注内容
```

### 2. 定义列表

```markdown
术语
: 定义内容
```

### 3. 折叠/详情块（GitHub）

```markdown
<details>
<summary>点击展开</summary>
隐藏内容
</details>
```

### 4. 数学公式（需 LaTeX）

```markdown
行内公式：$E=mc^2$
块公式：$$x = \frac{-b}{2a}$$
```

### 5. 高亮

```markdown
==高亮文字==
```

### 6. 上下标

```markdown
H~2~O      （下标）
X^2^       （上标）
```

## 十三、完整示例（综合）

```markdown
# 项目说明

> **重要**：请先安装依赖。

1. 克隆仓库
   `git clone xxx`

2. 安装依赖
   - [x] npm install
   - [ ] 配置环境变量

| 参数 | 说明 |
|------|------|
| `--port` | 端口号，默认 `3000` |
```
